# Thor Ragnarok ![Static Badge](https://img.shields.io/badge/Build%3A%20Completed-brightgreen)

<img width="1896" height="900" alt="image" src="https://github.com/user-attachments/assets/6a5287b1-c411-4cfe-8069-e5b3a9db85db" /> <br>


This project was developed by Gustavo Campelo during the limited event “A Nova Era - Projeto Thor”.

It is a landing page inspired by the universe of Thor: Ragnarok, created with a focus on immersive visual experiences, combining animations, video, and 3D elements to create a dynamic and cinematic navigation experience.

The page features a preloader system, a looped video in the hero section, an intermediate section with a video window, and an interactive experience where, as the user scrolls, the 3D Mjölnir hammer follows the user during scrolling, creating a depth and immersion effect. At the end, the project presents a modern and visually refined footer.

The entire design was initially developed in Figma, from the early structure to the definition of UI/UX, ensuring a consistent visual foundation before implementation.

During development, best practices in semantic HTML, SEO, and performance were applied, including image optimization and attention to the overall weight of the application to ensure fast loading.

In addition, the project incorporated artificial intelligence tools to accelerate the creative and visual production process. The hero and footer images were generated with the help of Gemini (BananaPro), and later animated in loop video format using the Flow tool. Despite the use of AI, all images and animations were manually adjusted and improved to ensure aesthetic and functional consistency.

The main 3D element of the project, the Mjölnir hammer, was obtained from the Sketchfab platform, optimized in Blender, and converted to .glb format with a focus on web performance. This was also the first practical experience with the Three.js library, which was used to integrate and manipulate the 3D model in the web environment.

## Features

- Initial preloader with smooth transition
- Hero section with autoplay loop video
- Video integration in an intermediate section
- Immersive scroll effect with 3D element (Mjölnir)
- Interactive 3D model using Three.js
- Smooth scroll-based animations with GSAP ScrollSmoother
- Modern footer with refined visual design
- Fully responsive layout
- Image optimization and performance-focused implementation
- Structure following SEO best practices and semantic HTML
